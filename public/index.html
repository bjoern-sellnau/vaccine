Configurable shims for modular JavaScript

Vaccine generates shims to make CommonJS (Node.js) or AMD modules work in the browser. Configurability makes the shims small by only including the required functionality. A small size is important for libraries (and some applications).

To use Vaccine, first set the options in the right panel. Next, copy vaccine.js and build.sh to your project's root directory. When you are ready to build your code, run sh build.sh > your_built_file.js. Note that all source code, and only source code, must be in a separate directory for the tiny build script to work.

For single file projects, Vaccine has the option to configure a UMD template, which doesn't need a build.
